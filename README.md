## このポートフォリオについて

本ポートフォリオでは、実案件をベースにした AWS サービスに関するトラブルシューティング事例や技術仕様の調査内容をまとめています。
AWS 公式ドキュメントおよびベストプラクティスに沿って原因の特定と解決策を整理しており、以下の2種類のケースを掲載しています：

- 自力で問題を切り分け・解決したケース
- AWSサポート へエスカレーションし、協力して解決したケース
